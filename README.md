# Facebook Clone

Hello to Facebook Clone! This is a full-featured project replicating core Facebook functionality with modern tech.

## 🚀 Quick Start

1. Clone the repo:
git clone https://github.com/yourusername/facebook-clone.git cd facebook-clone


2. Install dependencies:
npm install


3. Set up environment variables (e.g., DB creds, JWT secret, Cloudinary keys) in `.env`.

4. Run the app:
npm run dev

Visit [http://localhost:3000](http://localhost:3000).

## 📱 Features

- User authentication & profiles
- News feed with posts, likes, comments
- Real-time notifications
- Friend requests & connections
- Photo uploads
- Create posts from user profile or page
- Create pages
- Follow users or pages
